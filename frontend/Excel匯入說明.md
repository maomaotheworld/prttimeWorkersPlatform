# Excel 匯入功能說明

## 功能特色

### 僅電腦版顯示

- 匯入 Excel 按鈕只在電腦版（非手機版）顯示
- 提供更好的桌面端操作體驗

### Excel 格式要求

| 欄位 | 說明       | 範例                   |
| ---- | ---------- | ---------------------- |
| A 欄 | 工讀生編號 | W001, W002, W003       |
| B 欄 | 姓名       | 張小明, 李小華, 王小美 |
| C 欄 | 組別名稱   | 前台組, 後台組, 清潔組 |
| D 欄 | 樓層       | 1F, B1, 2F, 3F         |
| E 欄 | 基本時薪   | 200, 220, 180          |

### 自動功能

1. **組別自動歸納**
   - 系統會自動收集所有不重複的組別名稱
   - 自動建立不存在的組別
   - 自動將工讀生分配到對應組別

2. **數據驗證**
   - 檢查必要欄位（編號、姓名）
   - 檢查編號是否重複
   - 驗證薪資格式

3. **預覽功能**
   - 匯入前預覽數據
   - 顯示有效/錯誤數據統計
   - 顯示將新建的組別數量

### 預設值設定

匯入時會自動設定以下預設值（用戶可後續修改）：

- 性別：男（用戶可後續修改）
- 等級：1（用戶可後續修改）
- 基本工時：8 小時（用戶可後續修改）

### 支援格式

- .xlsx (Excel 2007+)
- .xls (Excel 97-2003)
- 檔案大小限制：5MB

## 操作步驟

1. 點擊「匯入 Excel」按鈕
2. 選擇或拖拽 Excel 檔案
3. 系統自動解析並顯示預覽
4. 確認數據無誤後點擊匯入
5. 系統自動建立組別和工讀生

## 範例 Excel 檔案

```
| A    | B      | C      | D   | E   |
|------|--------|--------|-----|-----|
| 編號 | 姓名   | 組別   | 樓層 | 時薪|
| W001 | 張小明 | 前台組 | 1F  | 200 |
| W002 | 李小華 | 前台組 | B1-1| 220 |
| W003 | 王小美 | 後台組 | B1  | 180 |
| W004 | 陳小強 | 清潔組 | 3F  | 190 |
```

匯入後會：

- 建立 3 個組別：前台組、後台組、清潔組
- 建立 4 名工讀生並分配到對應組別
- 設定各自的樓層資訊
- 設定各自的基本時薪
